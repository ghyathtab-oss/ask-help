 `true` فقط يظهرون في الصفحة الرئيسية.

## ملاحظات مهمة عن SMS

- الموقع لا يرسل الرسائل بنفسه ولا يستخدم API.
- زر الإرسال يفتح تطبيق الرسائل الافتراضي في الهاتف عبر رابط `sms:`.
- المستخدم يراجع الرسالة ويضغط **إرسال** بنفسه.
- يختلف سلوك الرسائل الجماعية قليلًا بين Android وiPhone حسب تطبيق الرسائل وإعدادات الجهاز.
