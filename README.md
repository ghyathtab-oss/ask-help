# نظام إذن الانصراف الجماعي — الجراحة العامة

تطبيق ثابت يتيح للمستخدم البحث واختيار أي عدد من المقيمين ثم فتح تطبيق الرسائل في الهاتف مع المستلمين ونص الرسالة جاهزين.

## تشغيل محلي

بسبب استخدام JavaScript Modules، شغّل المشروع من خادم محلي صغير بدل فتح `index.html` مباشرة:

```bash
python -m http.server 8080
```

ثم افتح `http://localhost:8080`.

## النشر

1. أنشئ مستودع جديدًا.
2. ارفع جميع الملفات كما هي إلى جذر المستودع.
3. فعّل النشر من الفرع الرئيسي ومجلد الجذر.
4. سيظهر رابط الموقع بعد اكتمال النشر.

## البحث والاختيار المباشر

الصفحة الرئيسية تعرض جميع المقيمين، ويمكن البحث بالاسم أو الرقم أو مستوى PGY ثم اختيار المقيم مباشرة.

## تفعيل المناوبين (اختياري)

تعديل القائمة يتم من الملف التالي فقط:

```text
data/residents.js
```

مثال لتفعيل مقيم كمناوب:

```js
{
  // ...
  dutyStatus: "مناوب",
  isOnDuty: true,
}
```

المقيمون الذين قيمة `isOnDuty` لديهم `true` يظهرون أولًا في الصفحة الرئيسية، لكن جميع المقيمين يبقون قابلين للبحث والاختيار.

## ملاحظات مهمة عن SMS

- الموقع لا يرسل الرسائل بنفسه ولا يستخدم API.
- زر الإرسال يفتح تطبيق الرسائل الافتراضي في الهاتف عبر رابط `sms:`.
- المستخدم يراجع الرسالة ويضغط **إرسال** بنفسه.
- يختلف سلوك الرسائل الجماعية قليلًا بين Android وiPhone حسب تطبيق الرسائل وإعدادات الجهاز.

## التخزين

- الاختيارات ونص الرسالة الافتراضي محفوظان محليًا باستخدام `localStorage`.
- لا يوجد Backend ولا تتبع.
- يمكن إضافة Firebase/Firestore لاحقًا لمزامنة النص بين الأجهزة، لكنه غير مطلوب.


## رمز الدخول

يطلب النظام رمز دخول عند فتح الموقع. الرمز الحالي هو `0000`.

> تنبيه: هذا حاجز واجهة بسيط فقط، لأن ملفات الموقع العامة يمكن رؤية محتواها داخل المتصفح. لا تستخدمه لحماية بيانات حساسة أو أرقام غير مصرح بنشرها.

لتغيير الرمز، عدّل هذا السطر في `app.js`:

```js
const ACCESS_PASSWORD = '0000';
```


## مصدر بيانات المقيمين

تم تضمين قائمة المقيمين مباشرةً داخل `data/residents.js` من ملف **مقيمين عليا.xlsx**. لا يحتاج الموقع إلى ملف Excel لتشغيله.
